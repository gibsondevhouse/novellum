# Gemini Roadmaps

This directory contains strategic development roadmaps for the Novellum project, generated by the Gemini CLI. These plans are designed to complement the existing `dev-docs/plans` and focus on long-term architectural health, stability, and modularity.

## Roadmap Index

1. **[01-unblocking-productivity.md](./01-unblocking-productivity.md)**: Immediate wins for developer experience and performance.
2. **[02-architecture-tightening.md](./02-architecture-tightening.md)**: Fixing boundary leaks and decoupling the data layer from the UI.
3. **[03-modularity-and-stability.md](./03-modularity-and-stability.md)**: Hardening the Svelte 5 migration, finalizing SQLite adoption, and ensuring desktop stability.

## How to use these Roadmaps

- **As a Developer**: Use these roadmaps to identify the next set of refactoring tasks.
- **As an Architect**: Refer to these documents when reviewing PRs to ensure new changes align with the long-term goal of a strictly modular, decoupled system.
- **With Gemini CLI**: You can ask Gemini to "implement stage X from roadmap Y" to begin execution of these tasks.
