## Behavior Added

- `calculateOutlineDiff()` compares an `OutlineDraft` against existing
  arc/act/chapter/scene hierarchy rows.
- The result reports:
  - root-first `insertions`
  - root-first `modifications` with changed field lists
  - leaf-first `deletions`
  - counts and `isEmpty`
- The engine detects parent changes as `parentId` modifications so moved acts,
  chapters, or scenes can be handled separately from insert/delete pairs.
