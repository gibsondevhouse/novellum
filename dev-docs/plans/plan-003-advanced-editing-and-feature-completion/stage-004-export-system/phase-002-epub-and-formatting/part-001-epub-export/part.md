## Objective

Add the EPUB driver using `epub-gen-memory`. Produce a valid `.epub` file with NCX table of contents, a cover placeholder page, and one HTML file per chapter. The file must open correctly in Calibre and Apple Books.

## Context

- `src/modules/export/services/assembler.ts` — assembled project data
- `src/modules/export/types.ts` — `ExportOptions`
- Install: `pnpm add epub-gen-memory`

## Target Files

| File                                            | Action                       |
| ----------------------------------------------- | ---------------------------- |
| `src/modules/export/services/epub-driver.ts`    | Create                       |
| `src/modules/export/services/export-service.ts` | Update — add `'epub'` branch |

## EPUB Structure

- Metadata: `title`, `author` (default "Unknown"), `genre` in description
- Cover: a simple HTML cover page with the book title centred (no image required)
- Chapters: one `content` entry per chapter; scenes concatenated as `<p>` tags separated by `<hr>`
- NCX: auto-generated by `epub-gen-memory` from chapter list
- `options.fontFamily` and `options.fontSize` injected as CSS in the chapter HTML template

## epub-gen-memory API

```ts
import Epub from 'epub-gen-memory';

const book = await Epub({
	title: project.title,
	author: 'Unknown',
	content: chapters.map((ch) => ({
		title: ch.title,
		data: scenesHtml,
	})),
});
// returns ArrayBuffer; convert to Blob
```
