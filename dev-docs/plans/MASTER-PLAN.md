# Master Plan Registry

Registry of all plans for the Novellum project. Plans are listed in execution order. Update this file whenever a plan is created, promoted, or archived.

## Status Key

| Status     | Meaning                            |
| ---------- | ---------------------------------- |
| `draft`    | Being designed, not yet executable |
| `active`   | Currently in execution             |
| `complete` | All stages finished and signed off |
| `archived` | Superseded or cancelled            |
