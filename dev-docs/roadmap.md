# Novellum — Roadmap

## Overview

The Novellum roadmap defines the staged development path from initial concept to a fully realized novel production system.

It ensures:

- focused development
- controlled scope
- meaningful progress
- a clear path to release

---

## Core Strategy

### 1. Build in Layers

Each phase adds a complete layer of functionality.

Do not skip layers.

---

### 2. Prioritize Usability Over Features

A smaller, stable system is better than a large, unstable one.

---

### 3. Validate Before Expanding

Each phase must be:

- usable
- testable
- stable

before moving forward.

---

## Phase 1 — MVP (V1)

### Goal

Create a functional, stable writing system with basic AI assistance.

---

### Core Features

#### Project System

- create project
- edit metadata
- basic dashboard

---

#### Story Bible

- create/edit characters
- basic relationships
- lore entries

---

#### Outliner

- create chapters
- define scene summaries
- basic structure editing

---

#### Drafting Editor

- chapter-based writing
- ONLYOFFICE integration
- autosave

---

#### AI (Basic)

- DraftAgent (scene drafting)
- BrainstormAgent (ideas)
- EditAgent (basic editing)

---

#### Export

- Markdown
- DOCX

---

### Success Criteria

- user can plan a novel
- user can draft chapters
- AI assists without breaking flow
- system is stable

---

## Feature Priority Order (V1 Focus)

1. Project system
2. Story bible (characters first)
3. Outliner
4. Drafting editor
5. AI drafting
6. Basic editing
7. Export

---

## What NOT to Build Early

Avoid in V1:

- complex AI chains
- full continuity engine
- plugin system
- collaboration
- advanced analytics

These add complexity without immediate value.

## Risks

- overbuilding too early
- unclear data relationships
- AI output inconsistency
- performance issues with large text
- UI complexity

---

## Mitigation Strategies

- follow spec-first workflow
- build incrementally
- enforce context discipline
- test with real writing scenarios
- refactor continuously

---

## Definition of Release (V1)

Novellum V1 is ready when:

- a user can create a project
- plan a story
- write multiple chapters
- use AI to assist drafting
- export a usable manuscript
- no major bugs block usage
