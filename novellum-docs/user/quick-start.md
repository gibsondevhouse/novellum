# Quick Start

> Last verified: 2026-05-07

Five minutes from install to your first saved scene.

## 1. Launch Novellum

Open the app. You'll land on the projects library.

## 2. Create a project

Click **New project**. Give it a title. Confirm.

## 3. Open the project hub

Each project has a **Hub** dashboard — your at-a-glance view of word count, recent scenes, and trust signals (when something needs attention).

## 4. Build your structure

From the project sidebar, open **Outline**. Novellum's hierarchy is:

```text
Arc → Act → Chapter → Scene → Beat
```

You don't need all five levels. A short story can live as a single chapter with a few scenes. Use what helps you.

Add an Arc, then an Act, then a Chapter, then a Scene.

## 5. Write your first scene

Click the scene to open the **Editor**. Type. Novellum autosaves continuously and takes periodic snapshots you can restore from later.

## 6. Configure AI (optional)

If you want AI assistance, head to **Settings → AI** and paste your OpenRouter API key. See [ai-setup.md](./ai-setup.md).

## 7. Back up your work

From **Settings → Data → Backup project**, save a `.novellum.zip` somewhere safe. You can restore from it at any time. See [backup-restore.md](./backup-restore.md).
