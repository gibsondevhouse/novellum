# Keyboard Shortcuts

Novellum supports a full set of keyboard shortcuts for editing, navigation, and application features. On Windows, replace **Cmd** with **Ctrl** unless noted otherwise.

---

## Editor Shortcuts

| Shortcut (macOS) | Shortcut (Windows) | Action | Context |
| :--- | :--- | :--- | :--- |
| **Cmd+B** | **Ctrl+B** | Bold | Editor — selected text |
| **Cmd+I** | **Ctrl+I** | Italic | Editor — selected text |
| **Cmd+U** | **Ctrl+U** | Underline | Editor — selected text |
| **Cmd+Z** | **Ctrl+Z** | Undo | Editor |
| **Cmd+Shift+Z** | **Ctrl+Y** | Redo | Editor |
| **Cmd+A** | **Ctrl+A** | Select all text in scene | Editor |
| **Cmd+F** | **Ctrl+F** | Find in scene | Editor |
| **Cmd+S** | **Ctrl+S** | Manual save (autosave is always on) | Editor |
| **Cmd+Shift+F** | **Ctrl+Shift+F** | Toggle Focus Mode (hide outline panel) | Editor |
| **Cmd+Shift+N** | **Ctrl+Shift+N** | Toggle Nova AI sidebar | Editor |
| **Cmd+E** | **Ctrl+E** | Open Export dialog | Project |
| **Escape** | **Escape** | Close modal / exit Focus Mode | Global |

## Outline Shortcuts

| Shortcut (macOS) | Shortcut (Windows) | Action | Context |
| :--- | :--- | :--- | :--- |
| **Cmd+Shift+Up** | **Ctrl+Shift+Up** | Move scene up in outline | Outline panel |
| **Cmd+Shift+Down** | **Ctrl+Shift+Down** | Move scene down in outline | Outline panel |

| **Delete** / **Backspace** | **Delete** | Delete selected node (confirmation required) | Outline panel |

---

## Notes

- **Autosave** is always active. **Cmd+S** / **Ctrl+S** triggers an immediate save, but your work is saved continuously regardless.
- **Focus Mode** hides the left panel (outline) and right panel (Nova sidebar) to give you a full-screen writing view. Press the same shortcut again to exit.
- Shortcuts that involve modifier combinations may conflict with operating-system shortcuts on some keyboards. If a shortcut does not work, check your OS accessibility or input settings.
