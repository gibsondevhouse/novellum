# Keyboard Shortcuts

> Last verified: 2026-05-07

Novellum is keyboard-friendly. The full shortcut list is configurable from **Settings → Shortcuts** (rebuild in flight under [plan-022-settings-ia](../../dev-docs/plans/plan-022-settings-ia/plan.md)).

The defaults below cover the most common surfaces. On macOS, ⌘ replaces Ctrl unless noted.

## Global

| Action | Shortcut |
| --- | --- |
| Open Nova | Ctrl + K |
| Toggle sidebar | Ctrl + B |
| Search project | Ctrl + Shift + F |

## Editor

| Action | Shortcut |
| --- | --- |
| Bold | Ctrl + B |
| Italic | Ctrl + I |
| Underline | Ctrl + U |
| Heading 1 | Ctrl + Alt + 1 |
| Heading 2 | Ctrl + Alt + 2 |
| Bullet list | Ctrl + Shift + 8 |
| Numbered list | Ctrl + Shift + 7 |
| Block quote | Ctrl + Shift + . |
| Undo | Ctrl + Z |
| Redo | Ctrl + Shift + Z |
| Save (manual) | Ctrl + S (autosave runs continuously) |

## Outline / hierarchy

| Action | Shortcut |
| --- | --- |
| New scene under current | Enter (when a scene/chapter is selected) |
| New sibling at this level | Shift + Enter |
| Move item up | Alt + ↑ |
| Move item down | Alt + ↓ |

## Notes

- Shortcuts respect your OS conventions (⌘ on macOS, Ctrl on Windows/Linux).
- Custom shortcuts and full reassignment are tracked under [plan-022](../../dev-docs/plans/plan-022-settings-ia/plan.md).
- If a shortcut conflicts with an OS-level binding, your OS wins. Reassign in Novellum.
